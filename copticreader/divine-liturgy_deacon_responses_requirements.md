# Divine Liturgy Requirements (Coptic Reader Edition)

## Scope

- Service: Divine Liturgy of St. Basil only.
- Source: copticreader.org.
- Audience: deacons needing concise, practical service prompts.

## Required Columns

1. Response Name / Section
2. English
3. Coptic
4. Arabic
5. Coptic English Transliteration

## Symbol Rules

- Use compact symbols in the response name or section row.
- Use symbols for location, candle, or censer only when they help the deacon act.
- Keep the main response text complete and uncut.

## Formatting

- Keep Coptic text in Unicode.
- Use CopticReaderUnicode font in generated Word files.
- Avoid expanding to non-essential variants.
