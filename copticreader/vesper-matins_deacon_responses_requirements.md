# Vespers and Matins Requirements (Coptic Reader Edition)

## Scope

- Services: Vespers and Matins only.
- Source: copticreader.org.
- Goal: compact practical deacon reference.

## Required Columns

1. Response Name / Section
2. English
3. Coptic
4. Arabic
5. Coptic English Transliteration

## Symbol Rules

- Use compact symbols in the response name or section row.
- Use symbols for location, candle, or censer only when they help the deacon act.
- Keep the main response text complete and uncut.

## Formatting Rules

- Keep Coptic as Unicode text.
- Keep entries concise.
- Keep output suitable for small page count in Word export.
